# Notice & Attribution

This project is an **independent, clean-room reimplementation** of a user story
mapping board, visually inspired by [storymaps.io](https://storymaps.io).

- No source code from storymaps.io was copied into this repository.
- The HTML, CSS, and JavaScript here were written from scratch.
- Only **factual colour values** (hex codes) and the general user-story-mapping
  layout convention were referenced to match the look and feel.
